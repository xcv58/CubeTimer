# Development

## Requirements
[`node.js`, `npm`](https://github.com/creationix/nvm#installation), and [`yarn`](https://yarnpkg.com/en/)

## Install

```bash
yarn
```

## Local development

```bash
yarn dev
```

## Local server

```bash
yarn start
```

## Deployment

```bash
yarn deploy
```

## Test

```bash
yarn test
```

Watch mode:

```bash
yarn test-watch
```
